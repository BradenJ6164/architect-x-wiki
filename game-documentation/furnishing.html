Furnishing

 

Furnishing is a vital tool used for placing items that can be interacted with by personnel or is a requirement for specific links.

As of now, there are 9 available pieces of furniture that each have a specific function or reason. 5 out of the 9 pieces of furniture has additional styles, adding up to a total of 14 different furniture.

To tell whether or not furniture can be swapped to a different variant, check to see if there is a gear button in the bottom right:

If there is, then the furnishing can be swapped out to a different variant using the large white arrows presented.

The Current list is as such:

  • The Kitchen Buffet, which can be switched into the Kitchen Counter. Both have the same use.
  • The Old Bed, which can be switched into the Old Bunk Bed. The Old Bed holds one test subject while the Old Bunk Bed holds two.
  • The Toilet, which can be switched into the Toilet Combo. Both have the same use.
  • The Bed, which can be switched into the Bunk Bed. The Bed allows one personnel to sleep while the Old Bunk Bed allows two.
  • The Cafeteria Table, which can be switched into a two-sided variant. Both have the same use.
  • The Standard Desk.
  • The Computer Desk.
  • The Metal Detector.
  • The Elevator.

Special Functions:

  • The Standard Desk and Computer desk can be used for SCP testing.
  • The Metal Detector checks for contraband (not implemented yet).
  • The Elevator can be used to transport personnel and test subjects up and down a floor.
  • The Computer Desk allows research to be done and is used for double profits.

If you wish to see more or are stuck on what furnishings can do, check out the Linking section to learn more.
