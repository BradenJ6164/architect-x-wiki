Zoning

Zoning is a core concept in Architect X, that defines your tycoon's areas. In order to have furnishing objects such as tables, chairs, beds, and desks they must be configured in a proper zone. You can delete and place as many zones as you need. You can also create zones in a convex-concave polygon. When a zone is placed, an interface shows the requirements for the zone to be valid. For any of the zone's objects to be functional, the zone must be valid. Be mindful that some zone types have a maximum number of objects, and those will be outlined below and presented in game.

Class D Cell

The Class D Cell will simply define a cell. You have the option to use an Old Bed or Old Bunk Bed. The Old Bunk Bed will provide two Class D slots. Upon successful completion of a zone, Class D's will automatically be issued to the foundation. See the Foundation Life Cycle for more information.

 

Safe Containment Chamber

The Safe Containment Chamber will define a location for Scientists to test on SCPs. Each chamber is limited to 1 SCP and 1 desk.

 

Laboratory Zone

The Laboratory zone will allow Scientists to research previously conducted tests to earn their profits 1.5x.
